## 📖 스토리

### 배경
- 주인공은 1년 전 키우던 고양이를 잃음
- 너무 슬프고 감정을 감당할 수 없어서 **억지로 감정을 묻어둠** (억지로 잊음)
- 억지로 잊으려 했지만, 감정이 해소되지 않아 **매일 무기력하고 우울함**
- 매일 회사 출근해서 배선 업무를 함

### 편지의 비밀
- 죽은 고양이가 **고양이별**에서 주인공을 다시 만나고 싶어함
- 죽은 자가 직접 다가갈 수 없어서 **편지로 소통**
- 편지는 **시간이 지나면 소멸**함 (야근하면 사라짐)
- **정시 퇴근**해서 편지를 잡으면 주인공의 손이 닿았기 때문에 **보존됨**

### 진엔딩 (숨겨진 요소)
- 편지 조각을 다 모으면 → 주인공이 눈치챔: **예전 고양이가 낙서한 그림과 똑같음 
- 자기가 키우던 고양이가 1년 전 죽었다는 걸 기억해냄.
- 그날 밤 **꿈**을 꿈
- 꿈에서 고양이를 만남 (사람이 잊어버리면 만날 수 없었으므로 그동안 못만났음)
- 제대로 인사 나누고, 고양이가 행복하게 지내는 걸 확인.
- 고양이는 곧 환생을 함. 환생 대기 순서가 끝났기 때문.
- 주인공은 일어나서 **펑펑 움**
- **여전히 슬프고 그립지만**, 억지로 묻어놨던 때보단 나음
- **감성적이고 희망적인 결말**

---
게임플레이 ↔ 스토리 연결 강화

예시:
- 배선 작업 = 주인공이 "감정선을 끊어놓고 사는 모습"의 은유
- 게임 초반엔 배선이 단순하고 기계적
- 편지를 받을수록 배선이 복잡해지고, 색이 생기고, 패턴이 유기적으로 변함
- = 감정이 돌아오는 과정을 시각적으로 표현


Day 1~N:
- 배선 퍼즐 클리어 → 퇴근
- 정시 퇴근 성공 → 편지 조각 1개 획득
- 야근 → 편지 못받음

편지 조각 다 모으면:
- "편지 맞추기" 버튼 활성화
- 클릭 → 진엔딩 컷신 시작

## 🎮 게임 메커니즘

### 코어 루프
1. **5단계 퍼즐** 진행 (배선 연결 업무)
2. **성공** → 정시 퇴근 → 우체통에서 **편지 조각 1개** 획득 (랜덤)
3. **실패** → 야근 → 편지 소멸 (획득 못함)
4. 게임 종료 → 다시 플레이 가능
5. 반복 플레이하며 편지 조각 수집
6. **모든 조각 모으면** → 진엔딩 해금 (숨겨진 요소)

### 편지 시스템
- **획득 방식**: 5단계 모두 성공 시 랜덤 편지 조각 1개
- **랜덤 알고리즘**: 확률 조정 (안 나온 조각 확률 증가)
- **진엔딩**: 플레이어는 처음엔 모름 (비밀)
- **편지 내용**: 고양이가 낙서한 것 (그림? 글자?)

### 난이도 곡선 (5단계)
- **1단계**: 전구 2개 (기본)
- **2단계**: 전구 3개
- **3단계**: 꽝 전구 1개 등장 (연결하면 안 됨)
- **4단계**: 전구 4개 또는 고정 타일 증가
- **5단계**: 최종 난관 (그리드 크기 증가? 복잡한 배치?)

## ✅ 결정된 사항

### 게임 플로우
- **한 판** = 5단계 클리어 → 편지 1개 획득
- 진엔딩은 **숨겨진 요소** (플레이어가 여러 번 플레이하며 발견. 처음엔 플레이어가 이 요소를 알 수 없다다)
- 실패 시 야근 → 편지 소멸 (다시 플레이해야 함)

### 스토리 & 톤앤매너
- 주제: 상실, 애도, 치유
- 톤: 따뜻하고 희망적이지만 슬픔도 있음
- 메시지: 슬픔을 억누르지 말고 마주하기

### 퍼즐 테마
- **배선 연결 퍼즐** (기존 MVP 활용)
- 5단계로 구성
- 난이도 점진적 상승

### 진엔딩 연출 방식
- **일러스트 시퀀스 + 배경음악**
- **언어 없음** (대사/자막 없이 시각적으로만 전달)
- 퍼즐 화면에서 일러스트가 순차적으로 페이드인/아웃
- 클릭 또는 자동 진행
